# Pong
Basic Pong made using Java-Swing

## Table of Content:
1. Controls
2. Modules Used
3. Pics

### Controls:-

W and S: Move the left Slider Up and Down <br/>
Up and Down Arrow: Move the right SLider Up and Down

### Modules Used:-

1. Swing: JFrame
2. awt: Color,Font,Graphics,Image,KeyAdapter,MouseAdapter
